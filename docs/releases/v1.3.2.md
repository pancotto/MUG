# MUG v1.3.2 Release Notes

Release date: 2026-05-12

## Summary

MUG v1.3.2 consolidates the synchronized phase-visibility workflow for three-phase graphs. The release focuses on consistency between UI and PDF output while preserving the existing ETL architecture, benchmark/profiling infrastructure, zoom/rebuild behavior, lazy loading, and Windows packaging flow.

## Highlights

- Added global phase synchronization across compatible three-phase graphs.
- Synchronized phase visibility between:
  - Tensão
  - Corrente
  - DHT Tensão
  - DHT Corrente
- Added session-level phase state for R, S, and T.
- Updated Plotly legend interaction so phase changes from any synchronized graph propagate to all compatible graphs.
- Preserved incremental updates using `Plotly.restyle`, avoiding full graph rebuilds on legend clicks.
- Added phase-aware Máx/Mín labels.
- Recalculated Máx/Mín markers using only currently visible phases.
- Preserved zoom/rebuild behavior while carrying the active phase state into rebuilt figures.
- Updated PDF export to reflect the current visible phases and recalculated Máx/Mín labels.
- Removed temporary phase-sync development logs from standard terminal output.

## Phase-Aware Labels

Máx/Mín labels now identify the originating phase or line pair:

- Corrente: `363,05 A (S)`
- Tensão: `386,25 V (ST)`
- DHT: `4,25 % (R)`

When a phase is hidden, Máx/Mín values are recalculated from the remaining visible phases instead of merely hiding markers visually.

## Compatibility

- Primata XLSX remains supported.
- Primata TXT remains supported.
- Embrasul TXT remains supported.
- The `ProcessedData` flow is preserved.
- Existing ETL behavior is preserved.
- Existing benchmark/profiling infrastructure is preserved.
- Lazy loading of PDF exporter and update checker is preserved.
- PyInstaller onedir build remains the supported executable packaging flow.
- Inno Setup remains the supported Windows installer flow.
